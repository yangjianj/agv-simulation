小车最短路径算法：    
1.找到小车当前邻点    
2.计算几个邻点到目标点的最短路径    
3.几个邻点路径+小车到邻点路径，取最小值   
4.计算path

2.模拟故障（停止在本地）    
3.运行过程中改变目标点    
4.在非site点时开始规划路径    
5.target为非site点    
6.运行过程中改变小车速度   
7.小车运行在两点直线旁边
